# rtimer

A timer made in electron

![Screenshot](./screenshots/sc_1.png)

# Features

- Play, pause, restart
- Manual time entry
- Minimize to tray

# Getting Started

```javascript
cd app
npm install
npm start
```

# Build
```
cd app
npm run dist
```
The installer will be found in `app/build`
